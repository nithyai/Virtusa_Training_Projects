# Library Management System (Java)

This project is a console-based Library Management System developed using Core Java. It helps automate basic library operations such as managing books, users, and transactions.

## Features

- Add, remove, and update books
- User registration system
- Issue books to users
- Return books with fine calculation
- Search books by title or author
- View all books and users

## Concepts Used

- Object-Oriented Programming (OOP)
- Classes and Objects
- ArrayList for dynamic data storage
- Scanner for user input
- Basic control structures (loops, conditionals)

## Project Structure

- Book class – stores book details
- User class – stores user details
- Transaction class – tracks issued and returned books
- Main class – handles menu and operations

## How to Run

1. Open terminal in the project folder
2. Compile the program:
   javac LibraryManagementSystem.java
3. Run the program:
   java LibraryManagementSystem

## Sample Functionalities

- Issue a book by entering book ID, user ID, and issue day
- Return a book and calculate fine if delayed (after 7 days)
- Search books using title or author name

## Tools Used

- Java (JDK 21)
- Visual Studio Code
